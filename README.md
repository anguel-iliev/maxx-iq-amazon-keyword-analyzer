# MAXX IQ – Amazon Keyword Analysis Software

## Project Overview
This project is a software prototype (web application) developed under an innovation project.
The goal is to extract and process large data sets from the Amazon platform for keyword analysis.

## Current Technology Readiness Level
Current TRL: 3–4 (Research, proof of concept, early prototype)

## Implemented Features (Prototype Stage)
- Project architecture and structure
- Keyword data model
- Initial Amazon data extraction logic (prototype)
- Ranking position calculation logic (draft)
- PPC cost estimation model (draft)

## Planned Features
- Full keyword ranking analysis
- Sales volume estimation per keyword
- Product niche analysis
- Web-based user interface
- Testing in real operational environment (TRL 7)

## Project Status
Early development / prototype phase.
